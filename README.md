# Webhook Server LumiereGYN

Servidor Node.js com Express para receber webhooks da SURI em /conversa.

## Instalação

```bash
npm install
```

## Configuração

Crie um arquivo `.env` com as variáveis necessárias.

## Uso

```bash
npm start
```

O servidor irá iniciar na porta definida em `PORT` ou 3000.
